About

Dies ist ein Demonstrator für das Zusammenspiel einer HTML-Fontseite mit Javacode und einem Rechenserver im Hintergrund.

  • Die Puzzleteile, das Brett und das Webdesign wurden mit Figma erstellt
  • Das Frontend mit Webflow. Die Interaktionen im Frontend sind mit Javascript implementiert
  • Das Backend ist in Python geschrieben und verwendet Django
  • Die Seiten laufen mit Apache auf Linux-Servern, die über die Azure Cloud zur Verfügung gestellt werden

Das Spiel IQ-Link ist ein Taschenpuzzle. Ziel ist es, 12 Puzzleteile auf dem Brett zu platzieren.
